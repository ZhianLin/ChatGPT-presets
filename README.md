# ChatGPT-presets
🌈  ChatGPT的一些预设包：DAN机器人、猫娘、丁真、律师、诗人、辩论家、小说家等等

## 项目使用

### 登录 [chatgpt](https://chat.openai.com/)，在对话前输入预设内容即可，[猫娘](https://raw.githubusercontent.com/tinygeeker/ChatGPT-presets/main/catgirl.txt)预设使用预览
